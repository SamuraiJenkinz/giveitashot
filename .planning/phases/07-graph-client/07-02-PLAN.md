Add the `send_email()` method to the `GraphClient` class in `src/graph_client.py`. Follow RESEARCH.md Pattern 3 exactly.

**Method signature** — must match `ews_client.py` EWSClient.send_email() signature exactly:
```python
def send_email(
    self,
    to_recipients: list[str] | str,
    subject: str,
    body_html: str,
    cc_recipients: list[str] | None = None,
    bcc_recipients: list[str] | None = None,
) -> None:
```

**Implementation:**
1. Normalize `to_recipients` to list if string
2. Filter empty strings from all recipient lists: `[r for r in to_recipients if r]`, same for cc and bcc
3. Raise `GraphClientError("At least one TO recipient is required")` if to_recipients is empty after filtering
4. Define `_make_recipient(addr)` helper returning `{"emailAddress": {"address": addr}}`
5. Build message dict:
   - `subject`: the subject parameter
   - `body`: `{"contentType": "HTML", "content": body_html}`
   - `toRecipients`: list of `_make_recipient()` dicts
   - Add `ccRecipients` only if cc_recipients is non-empty
   - Add `bccRecipients` only if bcc_recipients is non-empty
6. SendAs support: `send_from = Config.get_send_from()` — if `send_from` and `send_from != Config.SENDER_EMAIL`, add `"from": _make_recipient(send_from)` to message dict
7. Build payload: `{"message": message, "saveToSentItems": True}` — boolean True, not string
8. URL: `f"{BASE_GRAPH_URL}/users/{Config.SENDER_EMAIL}/sendMail"`
9. Call `self._make_request("POST", url, json=payload)`
10. Check `response.ok` — if not, call `self._raise_graph_error("send email", response)`
11. Log at INFO level: `f"Email sent successfully to {len(to_recipients)} recipient(s)"`

**Logging** (match EWSClient pattern):
- Before sending: log sender, recipients (TO/CC/BCC), SendAs info if applicable
- After success: "Email sent successfully"

IMPORTANT: `sendMail` returns 202 Accepted with empty body — use `response.ok` (True for 2xx), NOT `response.status_code == 200`.
