5. Create `GraphAuthenticator` class (replaces EWSAuthenticator):
   - Same `__init__` as before: calls `Config.validate()`, initializes `self._app = None`. Remove `self._access_token` (unnecessary state).
   - Same `app` property (lazy init of ConfidentialClientApplication using Config.CLIENT_ID, Config.CLIENT_SECRET, Config.get_authority()).
   - `get_access_token()` method: Use ONLY `self.app.acquire_token_for_client(scopes=GRAPH_SCOPE)` — do NOT call `acquire_token_silent` first (redundant since MSAL 1.23, see RESEARCH.md). Return `result["access_token"]` as plain string on success. Raise `AuthenticationError` with error/error_description on failure.
   - `get_ews_credentials()` method (BACKWARD-COMPAT SHIM): This method preserves the contract that main.py line 154 depends on. Implementation:
     ```python
     def get_ews_credentials(self) -> OAuth2Credentials:
         """
         Get EWS credentials for use with exchangelib.
         BACKWARD-COMPAT SHIM — kept so main.py continues to work until Phase 8.
         Remove when main.py migrates to Graph client.

         Returns:
             OAuth2Credentials: Credentials object for exchangelib.
         """
         self.get_access_token()  # Validates auth works (uses Graph scope now)

         # Create OAuth2 credentials for exchangelib (app-only / client credentials)
         credentials = OAuth2Credentials(
             client_id=Config.CLIENT_ID,
             client_secret=Config.CLIENT_SECRET,
             tenant_id=Config.TENANT_ID,
             identity=Identity(primary_smtp_address=Config.USER_EMAIL)
         )
         return credentials
     ```
     IMPORTANT: This method calls `get_access_token()` first to validate that auth is working (it will raise AuthenticationError on failure), then constructs the OAuth2Credentials object the same way the old EWSAuthenticator did. The OAuth2Credentials object is what exchangelib needs — exchangelib will internally acquire its own EWS-scoped token. This is the same pattern as the existing code.
   - `clear_cache()` method: Set `self._app = None`, log message.

6. Add backward-compat alias at module level after the class: `EWSAuthenticator = GraphAuthenticator  # Alias — remove when main.py is updated (Phase 8)`. This ensures `from .auth import EWSAuthenticator` in main.py continues to work.
