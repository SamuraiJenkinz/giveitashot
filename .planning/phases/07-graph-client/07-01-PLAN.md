Create `src/graph_client.py` with the following components:

**1. Imports and constants:**
- `import logging, time, re, requests` and `from html import unescape`, `from dataclasses import dataclass`, `from datetime import datetime, timezone`, `from typing import Optional, Any`
- `from .config import Config`

- `SELECT_FIELDS = "id,internetMessageId,subject,sender,from,body,receivedDateTime,hasAttachments"`

**2. Email dataclass** — copy EXACTLY from `ews_client.py` (lines 30-54). Same fields, same types, same `received_time_local` and `is_major_update` properties, same `classification: Optional[Any] = None`. This preserves the downstream contract byte-for-byte.

**3. GraphClientError exception class** — mirrors `EWSClientError` pattern.

**4. GraphClient class with `__init__`:**
- Constructor takes `authenticator` parameter (a `GraphAuthenticator` instance, but type-hint loosely to avoid circular import — just document it expects `.get_access_token() -> str`)
- Store `self._authenticator = authenticator`
- Create `self._session = requests.Session()` — enables clean mock patching in tests

**5. `_make_request(self, method: str, url: str, **kwargs) -> requests.Response`:**
- Implements the retry loop from RESEARCH.md Pattern 1
- On each attempt: call `self._authenticator.get_access_token()` to get fresh token, set `Authorization: Bearer {token}` and `Content-Type: application/json` headers
- On 401 and attempt < MAX_RETRIES: log warning "Token may have expired, re-authenticating", continue (silent re-auth via MSAL)
- On 429/502/503/504 and attempt < MAX_RETRIES: parse `Retry-After` header as int (fallback `2 ** attempt`), log at INFO level "Throttled by Graph API, retrying in {n}s (attempt {i}/{MAX_RETRIES})", sleep, continue
- On success (any other status): return response
- After loop exhausted: raise `GraphClientError(f"Request failed after {MAX_RETRIES} attempts: {url}")`
- Use `self._session.request(method, url, headers=headers, **kwargs)` for the actual HTTP call

**6. `_raise_graph_error(self, operation: str, response: requests.Response) -> None`:**
- Extract Graph error details from JSON response body: `response.json().get("error", {})` -> get "code" and "message"
- Raise `GraphClientError(f"Failed to {operation}: {response.status_code} {response.reason} - {detail}")`
- Wrap JSON parsing in try/except — fall back to `response.text` if JSON parsing fails

**7. `_strip_html(self, html_content: str) -> str`:**
- Copy VERBATIM from `ews_client.py` lines 111-121. Same regex, same unescape, same whitespace normalization.

**8. `_parse_datetime(self, dt_str: str) -> datetime`:**
- Handle empty string: return `datetime.now(timezone.utc)`
- Use `datetime.fromisoformat(dt_str.replace("Z", "+00:00"))` for Python 3.9/3.10 compat
- Wrap in try/except ValueError — log warning, return `datetime.now(timezone.utc)` on failure

**9. `_parse_message(self, msg: dict) -> Email`:**
- Follow RESEARCH.md Pattern 4 exactly
- `id`: use `msg.get("internetMessageId") or msg.get("id", "")` — internetMessageId is stable across folder moves
- `subject`: `msg.get("subject") or "(No Subject)"`
- `sender`: extract from `msg.get("sender") or msg.get("from") or {}` -> `sender.get("emailAddress", {})` -> name and address with "Unknown"/"unknown@unknown.com" defaults
- `body`: get `msg.get("body", {}).get("content", "")`, pass through `self._strip_html()`
- `body_preview`: `body_content[:200]`
- `received_datetime`: pass `msg.get("receivedDateTime", "")` to `self._parse_datetime()`
- `has_attachments`: `bool(msg.get("hasAttachments", False))`

**10. `get_shared_mailbox_emails(self, shared_mailbox: str, since: datetime | None = None, max_emails: int = 100) -> list[Email]`:**
- Follow RESEARCH.md Pattern 2 exactly
- Default `since` to today at midnight UTC if None
- Format since as `since.strftime("%Y-%m-%dT%H:%M:%SZ")` — NO quotes in OData filter
- Build params dict: `$filter`, `$select` (using SELECT_FIELDS), `$orderby: "receivedDateTime desc"`, `$top: min(max_emails, 100)`
- CRITICAL: `$orderby` property (receivedDateTime) MUST appear in `$filter` — this is already satisfied
- Initial URL: `f"{BASE_GRAPH_URL}/users/{shared_mailbox}/messages"`
- Pagination loop: while url and len(all_emails) < max_emails
- Call `self._make_request("GET", url, params=params)` — but only pass params on the FIRST request. Set `params = None` after first iteration (nextLink already contains all params)
- Check `response.ok` — if not, call `self._raise_graph_error("retrieve emails", response)`
- Parse `data.get("value", [])` — for each msg, try `self._parse_message(msg)`, catch Exception, log warning, skip
- Get next URL from `data.get("@odata.nextLink")` — None when last page
- Return `all_emails[:max_emails]`
- Wrap entire method in try/except: catch `GraphClientError` and re-raise, catch generic Exception and wrap in `GraphClientError`
